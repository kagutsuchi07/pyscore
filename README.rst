pyscore
=====

pyscore is a simple script for managing soccer scores.
It is written entirely in Python.


Usage
-----

.. code-block:: bash

    >>> pyscore insert
    1 1 Arsenal NULL Chelsea NULL NULL
    >>> pyscore update
    Fixture: 1 First Team Score: 1 Second Team Score: 2 ...UPDATED
    >>> pyscore update
    OK
